## Overview
This is my B.Tech Minor Project developed using Arduino. The system records important vehicle events and displays information on an OLED display to improve vehicle monitoring and safety.

## Objective
To design a smart vehicle event data recorder that stores important events and provides real-time information.

## Technologies Used
### Hardware
- Arduino Uno (ATmega328P)
- OLED Display (SSD1306)
- DHT11 Temperature Sensor
- Vibration Sensor
- Fire Sensor
- Voice IC
- Buzzer
- Speaker
- LED
- Power Supply Circuit

### Software
- Arduino IDE
- Embedded C / Arduino Programming

- ## Working
- Detects vibration using the vibration sensor.
- Detects fire using the flame sensor.
- Measures temperature using the DHT11 sensor.
- Displays information on the OLED display.
- Gives voice and buzzer alerts.
- Stores important vehicle event data.

- ## Results
The Intelligent Event Data Recorder successfully detects important vehicle events such as vibration, fire, and temperature changes. The detected information is displayed on the OLED display along with voice and buzzer alerts.
  
- ## Features
- Records important vehicle events
- Displays information on OLED
- Stores event data
- User-friendly interface
