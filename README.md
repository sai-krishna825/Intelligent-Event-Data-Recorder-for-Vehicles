# Intelligent Event Data Recorder for Vehicles

## Overview
This is my B.Tech Minor Project developed using Arduino. The system records important vehicle events and displays information on an OLED display to improve vehicle monitoring and safety.

## Objective
To design a smart vehicle event data recorder that stores important events and provides real-time information.

## Technologies Used

### Hardware
- Arduino Uno (ATmega328P)
- OLED Display (SSD1306)
- DHT11 Temperature Sensor
- Vibration Sensor
- Fire Sensor
- Voice IC
- Buzzer
- Speaker
- LED
- Power Supply Circuit

### Software
- Arduino IDE
- Embedded C / Arduino Programming
  
- ## Features
- Records important vehicle events
- Displays information on OLED
- Stores event data
- User-friendly interface
